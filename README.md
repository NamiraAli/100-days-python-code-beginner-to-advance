# 🚀 About the Challenge

Welcome to my **100 Days of Python Coding Journey**.
In this challenge, I commit to coding **every day for 100 days** to improve my Python programming skills by building projects, solving problems, and learning new concepts.

This repository contains all the exercises, mini projects, and practice programs that I complete throughout the challenge.

---

# 🎯 Goals

* Practice Python programming daily
* Strengthen problem-solving skills
* Build small practical projects
* Learn important Python concepts and libraries
* Develop consistency in coding
